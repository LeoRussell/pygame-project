Для запуска игры через средства python, используйте файл main.py. Для запуска игры в обособленном режиме, используйте файл main.exe по пути ../game/main/main.exe.
 
Minicruft Dumgeons - карточная игра в жанре roguelike для персональных компьютеров.

В начале игры пользователю дается возможность выбора главного персонажа, что никоем образом не повлияет на дальнейший игровой процесс, а все изменения будут лишь косметическими. Для управления игрок может использовать стрелочки или кнопки WASD на клавиатуре. Нажатие ESCAPE в процессе игры вызывает меню выбора персонажей с возможностью смены образа без сброса прогресса в данном "забеге". При смерти игроку на экран будет выведена статистика "забега", а также будет предложено начать "забег" заново нажатием клавиши SPACE, или покинуть игру нажатием кнопки ESCAPE. 

В процессе игры игроку будут встречаться враги, которых он, для дальнейшего продвижения, должен уничтожать.
Урон для кажого из врагов, расчитывается по такому принципу: врагу наносится максимально возможный урон, равный количеству единиц оружия героя, и если враг от нанесенного урона погибает, игроку возвращаются единицы оружия по формуле кол-во очков оружия героя - кол-во очков здоровья врага. В случае, если очков оружия героя недостаточно для убийства врага, последний получает урон по обратной формуле. Если игрок не иммет очков оружия, урон врагу будет нанесен за счет очков здоровья с вычитом последних по указанным выше формулам, с поправкой на то, что в случае недостатка очков здоровья для убийства врага, герой погибнет.

По ходу игру игроку будут встречаться различные объекты разных типов и особенностей. По индикатору в левом нижем углу можно узнать, какой объект появится следующим.

Краткое описание каждого объекта в игре.
● Враги:
– Зомби, 6 единиц здоровья (3 полных сердца), является простым врагом, после смерти роняет кусочек золота.
– Зомби со шлемом, 7 единиц здоровья (3 полных сердца и 1 щит), после потери шлема превращается в обычного зомби.
– Скелет, 4 единицы здоровья (2 полных сердца), ничем особенным не выделяется, после смерти роняет кусочек золота.
– Голем, 8 единиц здоровья (4 полных сердца), является самым сильным мобом в игре, после смерти роняет алмаз.
– Крипер, 4 единицы здоровья (2 полных сердца), является простым врагом, после смерти роняет динамит с таймером в несколько (3-4) ходов.
– Эндермен, 4 единицы здоровья (2 полных сердца), является ультимативным врагом с возможностью телепортироваться каждый ход в случайную клетку, меняя находящийся на ней предмет и себя местами, двигать игрока не способен, после смерти роняет кусочек золота.
– Эндермит, 2 единицы здоровья (1 полное сердце), является слабым врагом, после смерти может телепортировать игрока в один из углов поля с заменой клетки данного угла.

● Пикапы (подбираемые предметы):
– Железный Меч, при подборе дает 5 единиц оружия (макс. 15).
– Алмазный Меч, при подброе дает 8 единиц оружия (макс. 15).
– Кусочек золота, при подборе дает 1 единицу золота (макс. 99).
– Алмаз, при подборе дает 3 единицы золота (макс. 99).
– Красная скляночка, при подборе дает 1 единицу здоровья (макс. 10).

● Сдвигаемые (С) и активируемые (А) объекты:
– Сундук (А), при взаимодействии роняет на своем месте случайным предмет из положительного пула.
– Плохой Сундук (А), при взаимодействии роняет на своем месте случайный предмет из отрицательного пула.
– Динамит [ТНТ] (С), при взаимодействии меняется с герояем местами, имеет таймер по количеству ходов, по истечению которых взрывается, уничтожая свою и рядом стоящие клетки. В случае героя наносит ему урон в 5 единиц.
